# Java Project 
- It's simple java project about Static Proxy which act as bridge between user & target class
- create  StudentProxy (Proxy) class that extends Student (Target) class
- user will first call method on the top of Proxy obj
- inside the method of Proxy , apply some logic/ non-functional code (logging , conditions , security)  & call the actual method on the top of the target class

### Prerequisites
- java 17
